Có ba cách viết CSS phổ biến khi xây dựng trang web:

CSS nội tuyến (Inline CSS): Viết trực tiếp trong thẻ HTML bằng thuộc tính style. Cách này giúp áp dụng nhanh nhưng làm mã nguồn bị rối và khó bảo trì, nên dùng khi demo, test thử


CSS nội bộ (Internal CSS): viết trong thẻ < style> bên trong file HTML. Thích hợp cho các trang web nhỏ hoặc cần tùy chỉnh riêng lẻ. Ưu điểm: có thể tái sử dụng Nhược điểm: không tối ưu cho các trang lớn hoặc khi dự án có nhiều trang html Nên dùng khi: cần ghi đè quy tắc của những thư viện CSS bên ngoài (phần này quan trọng)

CSS ngoài (External CSS): Viết trong file .css riêng và liên kết với HTML bằng thẻ . Đây là cách tối ưu, giúp tái sử dụng và quản lý dễ dàng. Trong đó, CSS ngoài là phương pháp được khuyến khích nhất để xây dựng giao diện chuyên nghiệp. ưu điểm: giúp file html và css gọn gàng, có thể tái sử dụng style Nhược điểm: nếu file CSS không tải được thì sẽ mất toàn bộ style Nên dùng khi: lúc nào cũng nên dùng ( và kết hợp cách 2(Internal) để ghi đè(override) nếu cần )
